شما یک Numpad دارید که میتوانید از آن برای نوشتن کلمات کمک بگیرید. numpad شما کلید های 0 تا 9 ای دارد که به هرکدام حروفی از 'a' تا 'z' نسبت داده شده. هیچ حرفی به دو کلید نسبت داده نشده است. یک نمونه از numpad آمده است:

0: a e u 1: b l v 2: c m w
3: d n x 4: i o y 5: f p z
6: g q 7: h r 8: k s
9: j t

از طرفی، یک دیکشنری از کلمات هم به شما داده شده. شما میتوانید از keypad برای نوشتن کلمات استفاده کنید. مثلا برای جمله "Hello World" باید کلید ها به شکلِ

"7 0 1 1 4 2 4 7 1 3"

وارد شوند. ممکن است یک دنباله از کلید ها بتواند نمایانگر چند عبارت از دیکشنری باشد یا حتی نمایانگر عبارتی از دیکشنری نباشد. میگوییم یک مجموعه از کلمات، یک دنباله از کلیدها را پوشش میدهد، اگر بتوان یک عبارت از آن کلمات ساخت که دنباله کلیدها نشان دهنده آن عبارت باشد. یک عبارت از به هم چسباندنِ چند کلمه ساخته میشود. حالا شما باید با گرفتنِ یک دیکشنری، یک numpad و یک دنباله از کلید ها، اندازه کوچکترین مجموعه ای از کلمات که دنباله کلیدها را پوشش میدهد، خروجی دهید.

ورودی:
پارامترهای تابع main که باید آن را کامل کنید.
Dictionary, Numpad, Keyset
توضیح بیشتر:

dictionary = ['worda','wordb', 'wordc']
numpad[i] = array of chars corresponding to i : [['a','b','c'] , ['e','f','g'] , ...]
Keyset = array of keys = [0,1,2,3,4,5,...]

محدودیت ها:

dictionary_size \(\le 10^4\)
Keyset_size \(\le 10^3\)

خروجی تابع باید اندازه کوچکترین مجموعه ای باشد که دنباله کلیدهای داده شده را پوشش میدهد.
ورودی و خروجی نمونه:

input:
dictionary = ["am","axe","exam","boy","colour","dam","dot","donkey","fox","new","prim","prime","primeval","pry","the","this","theory","van"]
numpad = [['a','b','c'],['d','e','f'],['g','h'],['i','j','y'], ['l','m','n'],['o','p','q'],['r','s'],['t','u'], ['v','w','x'],['k','z']]
keyset = [7,2,1,5,6,3,4,1,8,0,4]
output:
2

dynamic programming.
f[0] = 0;
f[i] = min{ f[j] + 1 }; for all 0 <= j < i and S[j+1 .. i] is a word.
